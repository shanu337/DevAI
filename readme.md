# DevAI 😈

![Python](https://img.shields.io/badge/Python-3.12-blue)
![CLI](https://img.shields.io/badge/CLI-Typer-green)

«AI-powered CLI tool to generate, explain, debug, and chat about code — directly from your terminal.»

---

## ⚡ Features

- 🧠 Generate code from prompts
- 📖 Explain code files
- 🐛 Debug errors instantly
- 💬 Chat with AI
- 🎨 Beautiful CLI output using Rich

---

## 🚀 Installation

pipx install .

---

## 🧠 Usage

devai generate "binary search in python"
devai explain main.py
devai fix "IndexError: list out of range"
devai chat "what is docker?"

---

## 🛠 Tech Stack

- Python 🐍
- Typer (CLI framework)
- Rich (beautiful terminal UI)
- Google Gemini API

---

## 🔐 Setup

Create a ".env" file in the root directory:

GEMINI_API_KEY=your_api_key_here

---

## 📸 Demo

devai generate "what is pipx?"
